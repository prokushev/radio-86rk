# Тесты МОНИТОРа и CP/M

## Общие сведения

Программа TEST предназначена для проверки на совместимость подпрограмм МОНИТОРа
и функций CP/M BIOS и BDOS. Она содержит некоторый контроль изменения РОН ЦП.

## Запуск программы

Программа может быть запущена как через директиву R монитора, так и с помощью
других средств.

Запуск директивой R МОНИТОРа выполняется следующими командами:

```
R????,????,100
G100
```
